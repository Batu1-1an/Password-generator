# Fancy Password Generator

A modern, web-based password generator application built with HTML, CSS, and JavaScript.

![Screenshot of the Password Generator](img/Password%20generator.png)

## Features

*   **Customizable Length:** Choose password length using a slider (8-32 characters).
*   **Character Type Selection:** Include/exclude character sets:
    *   Uppercase Letters (A-Z)
    *   Lowercase Letters (a-z)
    *   Numbers (0-9)
    *   Symbols (!@#$%^&amp;*...)
*   **Exclude Ambiguous Characters:** Option to remove characters like `I, l, 1, O, 0`.
*   **Password Strength Indicator:** Visual feedback (bar and text) on the generated password's strength.
*   **Copy to Clipboard:** Easily copy the generated password with a single click.
*   **Dark Mode:** Toggle between light and dark themes for comfortable viewing. Theme preference is saved locally.
*   **Modern UI:** Clean interface with smooth transitions.

## How to Use

1.  Clone or download this repository.
2.  Open the `index.html` file in your web browser.
3.  Adjust the settings (length, character types) to your preference.
4.  Click the "Generate Password" button.
5.  Click the "Copy" button to copy the password to your clipboard.
